# Cat Social Network
This is a [React](https://reactjs.org/) and [Nextjs](https://nextjs.org/) template to play around with!

You will need [Node.js](https://nodejs.org/en/) installed on your computer to use play around with this code.

When you have Node installed run the following commands on the command line:

1. `npm install`
2. `npm run dev`

The url of the site should show up. When you make changes, the site will be updated!
